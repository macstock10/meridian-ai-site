# Meridian AI Landing Page

Single-page Next.js App Router site for Meridian AI, a real estate-focused AI consulting business.

## Run locally

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

## Deploy on Railway

1. Create a new Railway project from this repo.
2. Railway auto-detects Node + Next.js.
3. Use default commands:
   - Build: `npm run build`
   - Start: `npm start`

No backend services or extra platform config required.
